# Struct

## Create a data type 
```c
typedef struct
{
    //Define attributes
    string name;
    string number;
}
//Define the name of the data type
person;
```
## Using a data type 
```c
//Create an array of persons
person people[n];

//Use the dot notation to access the attributes
people[0].name = "Matheus";
people[0].number = "+55 (62) 98468-3275"
```
