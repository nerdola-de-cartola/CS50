# Variables


# Incrementing variables

```c
variable = variable + 1;
variable += 1;
variable++;
```

# Boolean expressions

```c
if (x < y)
{
    printf("x is less than y\n");
}
else if (x > y)
{
    printf("x is greater than y\n");
}
else
{
    printf("x is equal to y\n");
}
```

# Loops

## While
```c
int i = 0
while(i < 50)
{
    printf("hello, world\n");
    i++;
}    
```
## Do while

```c
int i = 0
do
{
    printf("hello, world\n");
}
while(i < 50)
```

## For

```c
for(int i = 0; i < 50; i++)
{
    printf("hello, world\n");
}
```
